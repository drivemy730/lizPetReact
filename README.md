🐶 LizPet Project 🐱

LizPet is a petshop platform with vet services. This repository contains the React frontend for the application and a Python folder for small data analysis tasks.

📂 Repository Structure

react/ — Frontend built with React, providing an interactive UI for customers and staff.

python/ — Scripts for small data analysis, including sales trends, pet records statistics, and visualizations using pandas, matplotlib, and chart/image generation.

✨ Features

View and manage petshop inventory

Schedule vet appointments

Track pet medical records

Analyze sales or pet data with Python scripts

Generate charts and visualizations from data

Interactive React dashboards for the frontend

🛠️ Tech Stack

Frontend: React, Vite, Tailwind CSS

Data Analysis: Python, pandas, matplotlib
